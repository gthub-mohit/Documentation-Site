C Programming Language

C is a

  • Procedural programming language
  • Fast, close to hardware
  • Widely used in OS, embedded systems, compilers

Basic Structure of C program

  • #include <stdio.h>
  • int main()
  • {
  • printf("Hello, World!");
  • return 0;
  • }

Explaination

  • #include <stdio.h> → standard input/output
  • main() → program execution starts here
  • return 0; → successful execution

Data Types

  • int a = 10;
  • float b = 3.14;
  • double c = 3.14159;
  • char d = 'A';

Variables & Constants

  • int x = 5;
  • const float PI = 3.14;

This section is intentionally kept concise. Deeper notes will be added when revisited during problem solving and projects.

HTML (HyperText Markup Language)

HTML is a

  • Markup language for structuring web pages
  • Defines content, not styling or logic
  • Foundation of every website

Basic Structure of HTML Document


<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Page</title>
</head>
<body>
    <h1>Hello World</h1>
</body>
</html>

Explanation

  • <!DOCTYPE html> → Defines HTML5 document
  • <head> → Metadata (title, links, meta tags)
  • <body> → Visible page content

Common HTML Tags


<h1>Heading</h1>
<p>Paragraph</p>
<a href="#">Link</a>
<img src="image.jpg">
<div>Block Container</div>

CSS (Cascading Style Sheets)

CSS is used to

  • Style HTML elements
  • Control layout, spacing, colors, fonts
  • Make websites responsive

Basic CSS Syntax


selector {
    property: value;
}

Example


body {
    background-color: #0f1115;
    color: white;
    font-family: Inter, sans-serif;
}
